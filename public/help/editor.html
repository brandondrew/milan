You are using an editor designed to transform plain text into valid HTML that will conform to the design of the site. You'll see the plain text in the text area, with a preview of how it will look when changed to HTML. You can style your text either with the buttons on the tool bar (as if you were using a word processor) or by using some simple conventions collectively known as Markdown.

Markdown Help

Mark things for emphasis by putting asterisks around a word or phrase. (This is normally rendered in italics.)
*This* word will be emphasized.→This word will be emphasized.
Make text stand out (usually in bold letters) by putting double asterisks around it.
**This** word will be strong.→This word will be strong.
To start a new paragraph, simply enter two carriage returns where you want to separate the paragraphs.

This is one paragraph.

This is another paragraph.

→

This is one paragraph.

This is another paragraph.

To create a bulleted list, simply put asterisks before the text, with each bullet at the beginning of a new line, and followed by 3 spaces.
* This is item # 1
* This is item # 2		→
  • This is item #1
  • This is item #2
Create a numbered list exactly how you would expect.
1 This is the first item.
2 This is the second		→
  1. This is the first item
  2. This is the second item

For lots of gory details, see the official documentation for Markdown
